# IntroLab – Prática 3: Pêndulo

## Objetivos

- **Medir a aceleração da gravidade** a partir da frequência de oscilação de um pêndulo em baixa amplitude.
- **Aprender conceitos básicos de ressonância e dissipação** observando o comportamento do pêndulo ao longo do tempo.
- **Utilizar ferramentas de ajuste de curva** (Mathematica, Python, C, Excel) para ajuste dos dados experimentais.
- **Implementar um sistema de hardware** (impressão 3D) para marcação dos tempos por interrupção de feixe laser e para segurar o celular no pêndulo.
