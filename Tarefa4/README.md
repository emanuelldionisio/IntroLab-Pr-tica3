# Tarefa 4 – Sistema de Hardware (Impressão 3D e Sensor Laser)

Projetar e implementar o sistema físico de suporte e medição do pêndulo.

## Componentes

- **Suporte impresso em 3D**: peça para fixar o celular ao pêndulo, garantindo estabilidade durante as oscilações.
- **Sistema de interrupção de feixe laser**: sensor laser para detecção automática do instante em que o pêndulo passa pelo ponto mais baixo, possibilitando a medição precisa do período.

## Etapas

1. Modelar as peças no software de CAD (ex.: Fusion 360, FreeCAD, Tinkercad).
2. Imprimir as peças em impressora 3D (PLA ou ABS).
3. Montar o circuito com o sensor laser e o microcontrolador (ex.: Arduino).
4. Validar o funcionamento do sistema comparando com medições manuais.
